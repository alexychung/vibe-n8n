# New Client Intake

When someone fills out our intake form (it's a webhook from Typeform), I need
it to check if the email is already in our system. If it is, just update the
record. If it's new, create a new record and ping the #new-clients Slack channel
so the team knows.

Medium stakes — we don't want to lose leads but it's not the end of the world
if one gets delayed.
